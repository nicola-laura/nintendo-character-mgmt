### 🍄 **과제 내용 및 기간**
⏵ '직원들을 관리할 수 있는 관리자 서비스를 만들어 보세요.'  
⏵ 2024년3월11일 ~ 2024년3월24일

### ⭐️ **주요 내용**

1. 로딩 애니메이션 (CSS Animation)
2. 스크롤이 가능한 형태의 리스팅 페이지 + Infinite scroll
3. 프로필 페이지
4. 캐릭터 추가: 사진 등록/삭제, 데이터 입력, Local Storage
5. 캐릭터 검색
6. 반응형 (Media query)
7. 기타: 상대수치 사용, JavaScript DOM event
